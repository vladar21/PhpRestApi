СТРУКТУРА ПРОЕКТА

.htaccess - перенаправления всех запросов на "точку входа" - файл index.php.

Api.php - аобстрактный класс - контроллер, с описанием методов API и вспомогательных функций к нему (сериализация и т.д.)

MovieApi.php - реализация абстрактного API контроллера для сущности movies

Movie.php - модель, описание данных и формирование и выполнение запросов к базе данных

Db.php - класс-обертка над PDO, работа с базой данных

Папка createdb содержит файл createdb.php - нечто вроде миграции, создает и наполняет базу данных ПРОЕКТА

Папка images - содержит файлы с картинками, которые поступают через методы add и update, и удаляются в методах update и delete
